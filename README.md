# How to use 

  1. install node.js 
  ```
  $ curl -sL https://deb.nodesource.com/setup_4.x | sudo -E bash -
  $ sudo apt-get update
  $ sudo apt-get install nodejs

  ```
  2. install required package  
  ```
  npm install request 
  ```
  3. fill suitable parameters into scripts
  4. run script
  ```
  nodejs ./plat.js
  ```
